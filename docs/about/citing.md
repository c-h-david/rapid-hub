# Citing RAPID

## Citation Information

If you use RAPID in your research, please cite it using the following information:

### Software Citation Formats

#### APA Style
> David, C. H. (2025). *Routing Application for Programmed Integration of Discharge (RAPID)* [Computer software]. GitHub. https://github.com/c-h-david/rapid2

#### MLA Style
> David, Cedric H. *Routing Application for Programmed Integration of Discharge (RAPID)*. GitHub, 2025, github.com/c-h-david/rapid2.
